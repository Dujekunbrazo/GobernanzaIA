# Plan Formal

Ya tenemos la idea aterrizada y quiero generar el `plan.md` de esta iniciativa.

Usa la iniciativa activa y el `input de planificacion` ya preparado en esta
conversacion.

Genera o actualiza `plan.md` en la carpeta correcta.

El `plan.md` debe ser ejecutable, no narrativo. Como minimo:
- problema real y evidencia base
- alcance y no-alcance
- restricciones, supuestos y riesgos
- tramos o commits logicos
- write set orientativo por tramo
- validacion minima por tramo
- rollback coherente
- Definition of Done

Antes de seguir, confirma que iniciativa estas usando y luego continua con el
plan.
