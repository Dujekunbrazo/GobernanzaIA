# F4 Plan

El Ask ya está congelado. Quiero pasar a planificación.

Usa la iniciativa activa de esta conversación.

Toma como base el `handoff.md` y el `ask.md` de esa iniciativa y genera el
`plan.md` en la misma carpeta.

El `plan.md` debe ser ejecutable, no narrativo. Como mínimo:

- owners y wiring canónico explícitos si toca capability transversal
- commits o tramos lógicos derivados del plan
- write set orientativo por commit
- validación mínima por commit
- rollback coherente

Antes de seguir, confirma qué iniciativa estás usando y luego continua con el plan.
