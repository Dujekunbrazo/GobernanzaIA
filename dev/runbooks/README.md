# Runbooks (Estado Actual)

Esta carpeta contiene runbooks heredados de otro proyecto.

Regla vigente:
- Los runbooks heredados fueron movidos a `dev/records/legacy/runbooks/`.
- No son fuente operativa para este repo hasta adaptación explícita.
- El estado oficial se define en `dev/runbooks/REGISTRY.md`.

Uso permitido:
- consulta histórica
- extracción de ideas

Uso no permitido:
- ejecutar procedimientos heredados como si fueran válidos para el repo actual
- usarlos para aprobar gates del workflow actual

Para operar en este repo, usar:
- `AGENTS.md`
- `dev/workflow.md`
- `dev/guarantees/*.md`
- `dev/prompts/*.md`
