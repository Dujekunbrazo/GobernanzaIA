# Docs Policy (Hard)

Fuente canónica:
- `dev/policies/documentation_rules.md`

Reglas mínimas:
- `README.md` no se reescribe completo.
- Actualización incremental obligatoria.
- No inventar rutas/comandos/features/archivos.
- Todo debe ser verificable en el repo actual.

Si no se puede verificar una afirmación:
- bloquear avance
- marcar estado `BLOQUEADO`
